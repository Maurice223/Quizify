PROJET QUIZIFY

Documentation Technique du Backend & Endpoints API

1. Architecture & Technologies

Stack : Java Spring Boot 3, MySQL, Hibernate/JPA, Debian Linux.
Concept : Système de gestion de quiz avec moteur de calcul de score automatique et persistance des données.

2. Documentation des Endpoints

👤 Gestion des Utilisateurs

MéthodeEndpointDescription
POST/users/registerInscription (Email/User uniques)
POST/users/loginAuthentification simple (Username)
GET/users/allListe de tous les utilisateurs
GET/users/{id}Détails d'un compte précis
DELETE/users/{id}Suppression de compte

📁 Gestion des Catégories

MéthodeEndpointDescription
GET/categories/allListe toutes les thématiques
POST/categories/addAjouter une nouvelle thématiques
DELETE/categories/{id}Supprimer une thématique

📝 Gestion des Quiz & Questions

MéthodeEndpointDescription
GET/quiz/allListe complète (Quiz + Questions + Choix)
POST/quiz/addCréation de quiz (Cascade All)
DELETE/quiz/{id}Suppression totale (Quiz et contenu)

📊 Scores & Algorithmes

MéthodeEndpointDescription
POST/scores/calculate/{qId}/{uId}Calcul intelligent : Vérification et sauvegarde
GET/scores/user/{id}/historyHistorique complet des notes par utilisateur
GET/scores/allListe globale des performances
DELETE/scores/{id}Nettoyage de l'historique

🔔 Système de Notifications NOUVEAU

MéthodeEndpointDescription
POST/notifications/addBroadcast : Envoi d'une alerte à tous les utilisateurs
GET/notifications/allFlux de notifications (trié par date de publication)
DELETE/notifications/{id}Suppression d'une notification archivée
