Projet Quizify

Architecture Backend & API REST - Développé par Maurice

1. Présentation du Projet

Environnement : Debian Linux | Java Spring Boot | MySQL | Hibernate
Objectif : Créer une plateforme de quiz complète avec calcul automatique des scores.

2. Documentation des Endpoints

Utilisateurs

MéthodeEndpointDescription
POST/api/users/registerInscription (Email/User uniques)
POST/api/users/loginAuthentification simple

Scores & Algorithmes

MéthodeEndpointDescription
POST/api/scores/calculate/{qId}/{uId}Calcul intelligent du score
GET/api/scores/user/{id}/historyHistorique complet par utilisateur
DELETE/api/scores/{id}Suppression d'un score

3. Points Forts

  • Calcul Côté Serveur : Sécurité accrue (le client ne peut pas tricher sur son score).
  • Architecture Propre : Relations ManyToOne/OneToMany maîtrisées.
  • Gestion des Circularités : Utilisation de @JsonIgnore pour des réponses API fluides.
